# cra-monorepo-template

Monorepo example using create-react-app and shared component library structure with yarn workspaces

### Directory structure

```
.
└── cra-monorepo-template/
    ├── packages/
    │   ├── app/ (create-react-app)
    │   │   ├── src/
    │   │   └── package.json
    │   └── shared/ (shared component library)
    │       └── src/
    │       └── package.json
    ├── package.json
    └── yarn.lock
```
